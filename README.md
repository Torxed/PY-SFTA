# Python SFTA

Python Set File/Protocol Type Association Default Application Windows 10/11


## Features
* Set File Type Association.

## Basic Usage

##### Set Acrobat Reader DC as Default .pdf reader:
```powershell
python SFTA.py "Applications\\sublime_text.exe" ".txt"

```
